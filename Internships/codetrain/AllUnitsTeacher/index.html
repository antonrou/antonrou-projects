Instructions for teachers:
Each of the links below will take you to a seperate ___.HTML file (webpage) but the programming code for each is in a seperate ___.js file.

For example: The Example 1 in Unit A link will take you to a webpage whose file name is exa1.html. That webpage uses a javascript file known as exa1_intro.js. It is the ___.js file that you will need to access to see the code.

The letters next to each file name in the sidebar correspond to each unit respectively. For instance, "exa" corresponds to Unit 1 and "exb" corresponds to Unit 2.

For easiest access to all of these file, we recommend you create an account with Replit.com and then Fork (aka copy) this replit so you have full read/write access to your own copy. Then check back periodically for updates on this original version.
