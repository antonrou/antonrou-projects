Web Languages

HTML

HTML is short for Hypertext Markup Language. HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. HTML is a text-based approach to describing how content contained within an HTML file is structured. This markup tells a web browser how to display the text, images and other forms of multimedia on a webpage. Web Page can contains a series of connections to other pages called hyperlinks

CSS

Short for cascading style sheets, CSS are used to describe reusable styles for presenting documents written in a markup language.

Three ways to use CSS

  • External Style Sheet
  • Internal Style Sheet
  • In-line style

Javascript

Javascript ia an object-oriented computer programming language commonly used to create interactive effects within web browsers

Server side and Database Languages

The server-side languages such as PHP, Ruby, JAVA (jsp) etc. runs a scripting language in a web server. A user's request is fulfilled by running a script directly on the web server to generate dynamic HTML pages. They are usually used to provide interactive web sites that interface to databases or other data stores on the server.
